# 错误处理

对于我们打开文件的操作，开始的时候我们使用的是Create，也就是创建一个新的文件，现在我们使用OpenFile来进行操作

```go
file, err := os.OpenFile(filename, os.O_EXCL|os.O_CREATE, 0666) //0666为可读可写的文件
```
